## Introduction

![Screenshot of AEE text editor interface](/uploads/aee.png)

In a world obsessed with feature bloat, AEE stands as a monument to restraint. This isn't just another text editor — it's a philosophical statement against the tyranny of complexity.

## Why I Maintain This Fork

I maintain this fork because someone has to. The original AEE was perfect in its minimalism, but modern systems broke it. Rather than abandon it to obsolescence, I patched the build system and fixed compatibility issues while preserving its soul.

## What Makes AEE Special

What makes AEE special:

- **Zero learning curve**: Open a file, start typing. No modes, no hidden commands.
- **Predictable behavior**: What you see is what you get, every time.
- **Lightweight**: Starts instantly, uses minimal resources.

## Technical Changes

The changes I made were surgical:

- Updated CMake configuration for modern compilers
- Fixed ncurses compatibility across distributions
- Maintained exact behavioral compatibility

## Use Cases

This editor is for when you need to make quick edits without ceremony. When you're debugging a config file at 3 AM and don't want to fight your tools.

## Installation and Usage

Install (AUR):

```bash
yay -S aee-anoraktrend-git
```

Usage:

```bash
aee /path/to/file.txt
```

The source code is intentionally readable. You can understand the entire editor in an afternoon — something you can't say about most modern software.
